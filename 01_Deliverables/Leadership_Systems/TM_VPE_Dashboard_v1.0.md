# TM_VPE_Dashboard_v1.0

## Purpose

The Vice President Education (VPE) Dashboard tracks the educational health of the club and ensures meetings are properly staffed and prepared. It provides a clear view of the **speaker pipeline, meeting role coverage, and member progress through Pathways**.

The VPE should be able to answer quickly:

> Are meetings fully staffed and are members progressing through the education program?

---

## Speaker Pipeline

| Metric                      | Current | Target / Notes        |
| --------------------------- | ------- | --------------------- |
| Speakers Next Meeting       |         | Minimum 2             |
| Meetings Scheduled Ahead    |         | 3–4 weeks recommended |
| Open Speaker Slots          |         | ≤1 preferred          |

A healthy club maintains **consistent speaking opportunities** while keeping the schedule full several weeks ahead.
