# TM_SAA_Dashboard_v1.0

## Purpose

The Sergeant-at-Arms (SAA) Dashboard ensures that **meeting logistics, environment, and materials** are prepared so every meeting runs smoothly and professionally.

The Sergeant-at-Arms should be able to answer quickly:

> Is the meeting space ready and do we have everything needed to run a high-quality meeting?

---

## Meeting Setup

| Item                             | Status | Notes |
| -------------------------------- | ------ | ----- |
| Room / Venue Confirmed           |        |       |
| Room Layout Prepared             |        |       |
| Lectern / Speaking Area Ready    |        |       |
| Timing Lights or Timer App Ready |        |       |

Proper setup ensures meetings start on time and run smoothly.

---

## Materials Preparation

| Item                       | Status | Notes |

| Item                           | Status | Notes |
| ------------------------------ | ------ | ----- |

---

## SAA Executive Report (Typical 60-Second Update)

Example summary:

* Room confirmed and prepared for next meeting
* Ballots and guest sign-in sheets printed
* Timing equipment tested and ready

Short updates ensure executive meetings stay efficient while confirming meeting logistics are under control.
