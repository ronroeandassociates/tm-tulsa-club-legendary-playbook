# TM_Treasurer_Dashboard_v1.0

## Purpose

The Treasurer Dashboard provides a concise financial snapshot for the executive team. It ensures transparency in club finances, tracks dues collection, and highlights upcoming financial obligations.

The Treasurer should be able to answer quickly:

> What is the current financial health of the club and are all dues and obligations up to date?
