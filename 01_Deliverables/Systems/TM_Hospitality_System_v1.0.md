# Purpose

The Hospitality System defines how Tulsa Club provides refreshments and welcoming meeting environments for members and guests.

Hospitality is an important part of the club culture and contributes to:

- guest comfort
- networking opportunities
- community building
- event celebrations
- Open House success

The system ensures that refreshments are organized, predictable, and easy for volunteers to manage.

---

# Hospitality Philosophy

Tulsa Club hospitality follows four principles:

1. **Welcoming Environment**
  Guests should immediately feel comfortable and welcomed.
2. **Simple and Efficient**
  Hospitality should not create excessive workload for volunteers.
3. **Consistent Standards**
  Events should follow predictable hospitality practices.
4. **Budget Awareness**
  Refreshments should remain reasonable in cost.

---

# Standard Hospitality Events

Hospitality is typically provided during the following events:


| Event               | Hospitality Level     |
| ------------------- | --------------------- |
| Leadership Showcase | Light refreshments    |
| Speech Contests     | Light refreshments    |
| Club Anniversary    | Light refreshments    |
| Open House          | Enhanced refreshments |
| Christmas Party     | Expanded refreshments |


Regular weekly meetings usually **do not require hospitality** unless planned for special purposes.

---

# Preferred Vendor

Primary vendor:

**Sam’s Club**

Reasons:

- close proximity to the venue
- affordable bulk trays
- consistent product availability
- minimal preparation required

Using a consistent vendor simplifies event planning.

---

# Standard Event Hospitality Package

For most events with approximately **20–30 attendees**, the following items work well.


| Item            | Quantity |
| --------------- | -------- |
| Fruit tray      | 1        |
| Vegetable tray  | 1        |
| Cookie tray     | 1        |
| Sparkling water | 1 case   |
| Bottled water   | 1 case   |


Estimated cost range:

$50–$80

This package is appropriate for:

- contests
- anniversary meetings
- leadership showcase
- small guest events

---

# Expanded Hospitality Package

For larger events such as Open Houses or holiday meetings, additional items may be included.

Possible additions:


| Item                     | Purpose            |
| ------------------------ | ------------------ |

| Cider or hot chocolate   | seasonal events    |
| Additional cookie tray   | larger attendance  |

$75–$120 depending on attendance.

# Hospitality Setup

Hospitality tables should be placed near the **side or rear of the room**.

Guidelines:

- avoid blocking the speaking area
- keep refreshments accessible before and after meetings
- maintain clear walking space

Typical placement:

Entrance Area

Guest Sign-In Table

Hospitality Table

Audience Tables

Speaker Area

---

# Setup Procedure

The Hospitality Lead or volunteers should complete setup **15–20 minutes before the meeting begins**.

Setup tasks include:

- arranging food trays
- placing napkins and plates
- setting up drinks
- ensuring trash receptacles are available

Food should be available during pre-meeting networking.

---

# Cleanup Procedure

Cleanup responsibilities should be assigned before the meeting ends.

Tasks include:

- disposing of trash
- consolidating leftover food
- cleaning surfaces
- returning the room to venue standards

Cleanup should typically take **10–15 minutes**.

---

# Volunteer Roles

Hospitality responsibilities may include:


| Role               | Responsibility                     |
| ------------------ | ---------------------------------- |
| Hospitality Lead   | coordinates food planning          |
| Setup Volunteers   | prepare food tables                |
| Cleanup Volunteers | restore the room after the meeting |


Responsibilities should rotate among members when possible.

---

# Budget Considerations

Hospitality should remain reasonable and sustainable.

Guidelines:

- limit routine event hospitality to $50–$80
- larger events may spend slightly more
- potluck options reduce cost

The Treasurer should be informed if club funds are used.

# Common Failure Points


| Issue                 | Prevention               |
| --------------------- | ------------------------ |
| Too little food       | estimate attendance      |
| Food in speaking area | place tables along walls |
| Volunteers unclear    | assign roles early       |
| Cleanup neglected     | assign cleanup team      |
