# Purpose

The Website Operations System defines how the Tulsa Club website is maintained and used to support club operations.

The website functions as the club’s **primary public information hub** and plays a key role in:

- attracting guests
- communicating meeting details
- promoting special events
- reinforcing club credibility

This system ensures the website remains accurate, functional, and aligned with the club’s membership growth efforts.

---

# Platform

The Tulsa Club website is hosted on:

**FreeToastHost**

FreeToastHost provides Toastmasters clubs with a structured platform for managing:

- club information
- meeting schedules
- member communication
- event announcements

Because the platform is standardized, updates are relatively simple and require minimal technical expertise.

---

# Domains

The club uses the following domains:

- tulsatoastmasters.org
- tulsatoastmasters.club

These domains direct visitors to the club’s FreeToastHost website.

Domains should remain active and properly configured to ensure guests can easily locate the club online.

---

# Website Objectives

The club website serves five primary objectives:

1. Inform visitors about the club
2. Provide clear meeting information
3. Encourage guests to attend
4. Promote club events
5. Support membership inquiries

The site should remain simple, clear, and welcoming.

---

# Core Website Content

The following information must always remain current.


| Content Area        | Required Information             |
| ------------------- | -------------------------------- |
| Homepage            | club overview and guest welcome  |
| Meeting Information | time, day, and location          |
| Contact Information | officer contact or club email    |
| Guest Invitation    | encouragement for visitors       |
| Events              | upcoming contests or Open Houses |


Incorrect or outdated information discourages guest visits.

---

# Content Ownership

Primary responsibility for website content belongs to:

**Vice President Public Relations**

Supporting roles may include:

Vice President Membership  
President  
Secretary

Responsibilities may be shared but ownership must remain clear.

---

# Update Responsibilities

The website should be updated when the following changes occur:


| Change                  | Update Required     |
| ----------------------- | ------------------- |
| Meeting location change | update immediately  |
| Meeting schedule change | update immediately  |
| Officer team change     | update annually     |
| Event announcements     | update as needed    |
| Open House promotion    | update before event |


Prompt updates ensure guests receive accurate information.

---

# Monthly Website Check

The Vice President Public Relations should perform a brief review once per month.

Checklist:

- confirm meeting time and location
- verify officer information
- review event announcements
- test contact links
- confirm website accessibility

This review usually requires less than 10 minutes.

---

# Event Promotion

The website should promote major club events including:

- Open House meetings
- Speech contests
- Anniversary celebrations
- leadership events

Announcements should include:

- event description
- meeting date
- invitation for guests

This helps drive attendance.

---

# Guest Conversion Role

The website plays a significant role in the **Guest Experience System**.

Visitors often check the website before attending a meeting.

The homepage should clearly communicate:

- guests are welcome
- no experience is required
- visitors may attend without obligation

This reduces hesitation for first-time guests.

# Common Failure Points


| Issue                                       | Prevention                      |
| ------------------------------------------- | ------------------------------- |
| Outdated meeting information                | perform monthly review          |
| Broken contact links                        | test forms periodically         |
| Event pages not updated                     | assign responsibility           |
| Website neglected during officer transition | include in transition checklist |


---

# Success Indicators

The website operations system is successful when:

- guests can easily find meeting information
- visitors feel confident attending
- events are promoted effectively
- the website remains accurate and professional

---

# Officer Transition

During officer transitions the outgoing VP Public Relations should:

- confirm website access credentials
- review website update procedures
- identify pages requiring updates

This ensures continuity between officer teams.

---

# Review and Update

The website system should be reviewed annually during:

- officer transition
- summer planning for the new Toastmasters year

Updates may be made based on:

- membership growth needs
- website usage patterns
- lessons learned from guest feedback.
