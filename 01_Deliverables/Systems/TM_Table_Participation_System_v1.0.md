# Purpose

The Table Participation System defines how Tulsa Club Toastmasters uses table seating to increase engagement, participation, and guest comfort during meetings.

Because the venue uses **square tables with four seats**, this system transforms tables from passive seating into active engagement groups.

The system encourages:

- increased Table Topics participation
- stronger member connections
- improved guest integration
- greater meeting energy

---

# Venue Context

The Tulsa Club meeting venue uses:

- square tables
- four seats per table
- a layout that is not normally rearranged weekly

Rather than moving furniture each week, the club uses a **table-based engagement model** that works with the existing layout.

Members simply rotate their chairs toward the speaking area when necessary.

---

# Table Engagement Model

Each table acts as a **small participation group** during the meeting.

Typical table size:

4 participants

Ideal composition:

- experienced member
- newer member
- guest (when available)
- additional member

This creates a supportive environment for conversation and speaking.

---

# Table Roles

Tables may optionally assign informal roles to encourage participation.

Possible roles include:

| Role | Responsibility |
|------|---------------|
Table Host | welcomes guests at the table |
Table Topics Candidate | prepared to respond if the table is called |
Table Listener | notes interesting insights during the meeting |
Table Reporter | may share a brief reflection if requested |

These roles rotate naturally among members.

---

# Table Topics Participation

The Table Topics Master may select speakers **by table rather than by individual**.

Example approach:

| Round | Table |
|------|------|
1 | Table 1 |
2 | Table 3 |
3 | Table 2 |
4 | Table 4 |

Each table quickly selects a volunteer speaker.

Benefits:

- reduces hesitation
- encourages teamwork
- increases participation

---

# Table Discussion Technique

For certain prompts, the Table Topics Master may allow a brief table discussion.

Process:

1. Prompt is announced
2. Tables discuss for 20–30 seconds
3. One volunteer speaks

This method:

- reduces speaker anxiety
- helps guests feel supported
- improves answer quality

---

# Guest Integration

Tables are an effective tool for welcoming guests.

Best practices:

- place one guest per table when possible
- seat guests with experienced members
- encourage conversation before the meeting begins

Members should introduce themselves to guests at the table.

---

# Pre-Meeting Table Conversations

Before the meeting begins, members may engage in short table introductions.

Example prompt:

“Please introduce yourself and share one communication skill you want to improve.”

This helps guests feel included immediately.

---

# Table Rotation Strategy

To prevent cliques and increase connection, members should occasionally change tables.

Suggested rhythm:

- rotate seating every 3–4 meetings
- encourage members to sit with different people
- place guests at different tables

Rotation encourages stronger relationships across the club.

---

# Table-Based Networking

Tables create natural networking opportunities.

This is especially helpful for:

- guests
- new members
- Open House events

Members should use the table structure to encourage conversation and connection.

---

# Speaker Visibility

During speeches:

- members should rotate chairs toward the speaking area
- clear sightlines should be maintained

The speaking area should remain unobstructed.

---

# Table Topics Encouragement

Members at each table should encourage one another to participate.

Experienced members should support newer members by:

- suggesting volunteers
- offering encouragement
- applauding participation

This helps build confidence among new speakers.

---

# Advantages of the Table System

The table-based system provides several advantages:

| Advantage | Impact |
|-----------|-------|
Guest comfort | easier integration |
Participation | increased Table Topics involvement |
Networking | stronger member connections |
Meeting energy | more interactive environment |

Rather than limiting engagement, tables can become a powerful participation tool.

# Common Failure Points

| Issue | Prevention |
|------|-------------|
Tables become passive | encourage table discussions |
Guests sit together | distribute guests across tables |
Low Table Topics participation | call on tables rather than individuals |
Members remain in same seats | encourage periodic rotation |

# Success Indicators

The Table Participation System is successful when:

- Table Topics participation increases
- guests feel comfortable engaging
- members interact across tables
- meetings feel energetic and collaborative
